.. _bluetooth_central:

蓝牙：中央角色(之前称为：master主设备)
##################

概述
********

该示例通过扫描其它BLE(蓝牙低功耗)设备，并与首个信号足够强的设备建立蓝牙连接的方式，来演示最基本的BLE主设备的功能。


要求
************

* 运行在主机host上的BlueZ，或
* 支持BLE的开发板

编译与运行
********************
该例程可在Zephyr树的如下路径找到 :file:`samples/bluetooth/central` 

详情请参考 :ref:`bluetooth setup section <bluetooth_setup>` .
