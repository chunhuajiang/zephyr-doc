.. _application:

应用程序开发原语
##############################

本章从总体上描述如何创建、编译和运行一个 Zephyr 应用程序。

听众
--------

尽管任何人都可以使用这些原语，但是本指导的基本听众是：

* 编写应用程序的开发人员。

* 想要学习关键功能和用法的系统架构人员。

开始之前
----------------

* 确保您的 Linux 主机满足 :ref:`getting_started` 中所说的最小需求。

* 确保您的环境变量配置正确了。参考 :ref:`apps_common_procedures`。
